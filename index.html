Library yang Digunakan

Python untuk Analisis Spasial:

  • GeoPandas – Membaca dan mengolah data spasial (vektor).
  • Shapely – Analisis geometri spasial (misalnya titik dalam poligon, pemotongan geometri).
  • NetworkX – Analisis jaringan jalan menggunakan graf.
  • OSMnx – Mendapatkan data jaringan jalan dari OpenStreetMap.
  • Rasterio – Membaca data raster, seperti slope dari DEMNAS.
  • Rasterstats – Mengambil nilai raster pada titik atau poligon tertentu.
  • NumPy – Perhitungan numerik yang efisien.
  • Pandas – Manipulasi data tabular, seperti join dan filtering.
  • Matplotlib – Visualisasi data spasial dalam bentuk grafik.
  • Folium – Visualisasi data interaktif berbasis leaflet.

Library untuk WebGIS & Deployment:

  • Mapbox GL JS – Visualisasi spasial interaktif di browser.
  • Tailwind CSS – Styling antarmuka yang modern dan responsif.
  • Vercel – Platform untuk hosting dan deployment WebGIS secara cepat dan gratis.
